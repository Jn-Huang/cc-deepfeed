# 足球赛事动态

## Scope

全球主要足球赛事赛果与动态。

关注：
- 欧冠、英超、西甲、德甲、意甲、法甲的重要比赛结果与亮点
- 国际赛事（世界杯预选赛、欧洲杯、美洲杯等）进展
- 重大转会消息、教练变动、球员伤病等影响赛季走势的新闻
- 战术分析与关键数据（进球、助攻、积分榜变化）

## Skip

- 花边新闻
- 球员私生活
- 博彩赔率

## Writing Style

**Target: 400-600 words per entry (for Chinese: 600-900 characters).** Match reporting tone, data-driven and concise:

- **Scoreline + key moments**: Lead with the result, then the goals/assists that shaped it. Include minute marks for key events.
- **Standings impact**: Always contextualize the result — what does this mean for the table? Title race? Relegation? European qualification?
- **Tactical observations**: When relevant, note formation changes, key matchups, or tactical shifts that decided the game. Don't force this if the match was straightforward.
- **What's next**: End with upcoming fixtures that matter (next round, head-to-head, derby dates).
- **Stats**: Include possession, xG, shots on target when they tell a story (e.g., "dominated possession but created nothing" is worth noting).
